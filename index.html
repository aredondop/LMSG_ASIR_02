  • Disipador Un disipador es un instrumento que se utiliza para bajar la temperatura de algunos componentes electrónicos.
  • Disco Duro Un disco duro o unidad de disco rígido, HDD o SSD es un dispositivo de almacenamiento de datos.
  • Unidad Óptica Una unidad de disco óptico es la unidad de disco que utiliza una luz láser como parte del proceso de lectura o escritura de datos desde un archivo a discos ópticos.
  • RAM La memoria de acceso aleatorio (Random Access Memory, RAM) es una memoria de almacenamiento a corto plazo.
  • Procesador Es un componente del hardware dentro de un ordenador, teléfonos inteligentes, y otros dispositivos programables.
  • Placa Base Es una tarjeta de circuito impreso a la que se conectan los componentes que constituyen la computadora.
  • Fuente de Alimentación Es el dispositivo que convierte la corriente alterna (CA), en una o varias corrientes continuas (CC), que alimentan los distintos circuitos del aparato electrónico al que se conecta.
  • Ventilador de Caja Es un componente que se usa para disipar el calor generado por componentes eléctricos. Por lo general, el aire caliente es sacado desde el interior del dispositivo con los ventiladores.
  • Disquetera Una unidad de disquete o disquetera (FDD del inglés floppy disk drive) se utiliza para leer y escribir datos en disquetes.
